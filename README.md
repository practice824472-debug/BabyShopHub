# BabyShopHub - Baby Products E-Commerce Application

A **university-level final project** - A comprehensive, full-featured baby products e-commerce mobile application built with Flutter, Firebase, and Provider state management.

## 🎯 Project Overview

BabyShopHub is a complete e-commerce solution specifically designed for baby products. It features a fully-functional user platform and an admin dashboard for managing products, orders, and users. The application demonstrates professional mobile app development practices including authentication, real-time database operations, state management, and responsive UI design.

---

## 🏗️ Tech Stack

| Component | Technology |
|-----------|-----------|
| **Frontend** | Flutter (Dart) |
| **Backend** | Firebase (Auth, Firestore, Storage) |
| **State Management** | Provider |
| **Architecture** | MVC (Models / Controllers / Screens) |
| **Design System** | Material 3 |
| **Database** | Cloud Firestore |

---

## ✨ Key Features

### 👤 User Features
- ✅ **Authentication** - Email/password registration, login, forgot password
- ✅ **Product Browsing** - Browse, search, filter by category
- ✅ **Shopping Cart** - Add, update, remove items with quantity management
- ✅ **Checkout** - Multi-step checkout: address → payment → confirmation
- ✅ **Order Management** - View order history with real-time status tracking
- ✅ **Product Reviews** - Rate and review products, view reviews from other users
- ✅ **User Profile** - Edit profile, manage addresses, change password
- ✅ **Support** - FAQ section and contact form
- ✅ **Responsive Design** - Optimized for all screen sizes

### 🛠️ Admin Features
- ✅ **Dashboard** - View key metrics (total users, orders, revenue)
- ✅ **Product Management** - Add, edit, delete products, manage stock
- ✅ **Order Management** - View all orders, update status, track shipments
- ✅ **User Management** - View users, disable/enable accounts
- ✅ **Analytics** - Track sales and user statistics

## 📱 User Interface

### Main Screens
1. **Splash Screen** - Auto-login and role-based navigation
2. **Authentication** - Login, Register, Forgot Password
3. **Home** - Product grid with search and category filters
4. **Product Details** - Full product info with reviews and add-to-cart
5. **Shopping Cart** - Cart summary with quantity controls
6. **Checkout** - 3-step process (address → dummy payment → confirmation)
7. **Orders** - Order history and tracking timeline
8. **Reviews** - View and write product reviews
9. **Profile** - User profile management
10. **Support** - FAQ and contact form
11. **Admin Dashboard** - Comprehensive admin controls
